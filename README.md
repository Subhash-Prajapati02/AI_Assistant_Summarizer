# 🤖 AI Assistant with OpenAI API

A simple AI Assistant built using the OpenAI API that can chat with users and summarize text.

---

## 🚀 Features

* AI chatbot
* Text summarization
* Secure API key using `.env`
* Simple frontend (HTML, CSS)

---

## 🛠️ Tech Stack

* Python
* OpenAI API
* HTML, CSS

---

## 📂 Project Structure

```
project/
├── main.py
├── .env
├── .gitignore
├── templates/
├── static/
└── README.md
```

---

## ⚙️ Setup

### 1. Install dependencies

```
pip install -r requirements.txt
```

### 2. Add API key

Create `.env` file:

```
OPENAI_API_KEY=your_api_key_here
```

### 3. Run project

```
python main.py
```

---

## 🔐 Note

Do not share your `.env` file. It contains your API key.
